# Typst-шаблон отчётов СПбГУТ

Этот шаблон предназначен для оформления студенческих работ (лабораторных,
практических, курсовых). Шаблон использует язык разметки Typst, который делает
оформление документов проще, быстрее и надёжнее, чем в MS Word.

С помощью этого шаблона из текстового файла с Typst-разметкой получается готовый
PDF с идеальным оформлением: шрифты, таблицы, формулы, листинги и т.д.

Кстати, нейронки могут генерировать эту Typst-разметку. Можно написать промпт,
получить код, отрендерить в PDF — и отчёт готов.

## Установка Typst

Для компиляции Typst-разметки в PDF-отчёты можно воспользоваться [официальным
онлайн-редактором Typst](https://typst.app/signup) или скачать [компилятор
Typst](https://github.com/typst/typst/releases).

Подробная инструкция по установке доступна на [GitHub-странице проекта
Typst](https://github.com/typst/typst?tab=readme-ov-file#installation).

Работу по компиляции можно упростить с помощью [LSP-сервера
tinymist](https://github.com/Myriad-Dreamin/tinymist), который интегрируется в
любой удобный редактор с поддержкой LSP и предоставляет возможность компиляции
на лету, подсветку синтаксиса, а также множество других полезных функций. В
частности, для VS Code существует специальный
[плагин](https://github.com/Myriad-Dreamin/tinymist/blob/main/editors/vscode).
Для локальной же работы подойдёт любой текстовый редактор с возможностью запуска
компиляции через терминал.

Для получения более подробной информации ознакомьтесь с официальной
документацией на https://typst.app/docs.

## Как использовать

1. Скопировать файл `bonch.typ` в проект.
2. Создать основной файл, например `report.typ`, и импортировать шаблон.
3. Вызвать функцию `conf` с данными и написать содержимое работы.
4. Скомпилировать PDF.

### Пример минимального файла `report.typ` (скопируйте и адаптируйте):

```typst
#import "bonch.typ": conf
#show: conf.with(
    title: [
        Циклические вычислительные процессы. Вычисления по \
        рекуррентным формулам.
    ],
    type: "Лабораторная работа №1",
    faculty: "Кибербезопасности",
    department: "Программной инженерии и вычислительной техники",
    subject: "Технологии и методы программирования",
    speciality: "10.03.01 - Информационная безопасность",
    students: (
        "Резниченко И., ИКБ-41",
        "Джинчарадзе Л., ИКБ-41",
    ),
    teachers: (
        "ст. преп. Шуппе А.",
    ),
)
// Тут начинается содержание самого отчёта: текст, картинки, таблицы и т.д.
```

## Кастомизация

Шаблон не имеет множества параметров, чтобы избежать усложнения. Если нужно
изменить что-то (например, шрифт кода или интервал строк), изучите Typst и
добавьте правила в шаблон:

- Изменить шрифт кода: `#show raw: set text(font: "Consolas")`
- Изменить интервал: `#set par(leading: 1em)`
- Добавить курсив для подписей: `#show figure.caption: set text(style: "italic")`

Читайте документацию Typst: https://typst.app/docs. Если шаблон не идеально
соответствует требованиям — доработайте самостоятельно.
